# AppleStore - 175 commits version
AppleStore is simple ecommerce project,
developed for practice my knowledge of Spring framework

                Technologies:

**Backend:**
- Java 11
- Spring boot

**Dependencies:**
- Spring Data JPA
- Thymeleaf
- Spring security
- Spring web
- Lombok
- Maven

**Frontend:**
- Html + css

**Database:**
- MySQL

**Features:**
       GENERAL:
* Login Page
* Ability to login and logout
* Remember me function
* Homepage
* Drop down menu
* See username of logged user
* Register new user
* Delete user

* PRODUCT:
* Page to see product properties
* Find specific product
* Add new product
* Update product
* Delete product
* Add category
* Edit category

* ORDER:
* Add/delete order
* Add/remove product from existing order
